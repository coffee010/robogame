# Linux 速查

## 常用命令

```bash
pwd                 # 当前目录
ls -la              # 查看文件
cd path             # 进入目录
mkdir name          # 创建目录
cp a b              # 复制
mv a b              # 移动或改名
rm file             # 删除文件，慎用
nano file           # 编辑文件
cat file            # 查看文件
```

## 树莓派常用命令

```bash
sudo raspi-config
rpicam-hello
aplay -l
ls /dev
systemctl status ssh
```
