# 硬件笔记

## 接口纠正

- Camera Module 3 使用 CSI 摄像头接口。
- 树莓派屏幕通常使用 DSI 显示接口，HDMI 屏也很常见。
- 喇叭通常不使用 CSI。CSI 是摄像头接口。音频建议选下面之一：
  - USB 声卡 + 普通小喇叭，最容易调试。
  - I2S 数字功放模块，例如 MAX98357A，适合嵌入式成品。
  - HDMI 音频，适合临时测试。
  - 蓝牙音箱，不推荐作为机器人内置方案，延迟和连接稳定性较差。

## 树莓派型号建议

- Raspberry Pi 5：性能更充足，适合 MediaPipe、屏幕和音频同时运行。
- Raspberry Pi 4 4GB/8GB：也可做，但实时识别帧率和散热要控制。

## 舵机建议

多个舵机不要直接从树莓派 5V 引脚供电。建议：

- 舵机单独电源供电。
- 树莓派和舵机电源共地。
- 使用 PCA9685 这类 PWM 驱动板。
- 每个舵机设置最小/最大角度，避免机械结构卡死。
